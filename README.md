# Quran Malayalam Admin

Content Management System for Quran Malayalam built with Next.js 14, MongoDB, and Tailwind CSS.

## Features
- Role-based authentication (Creator/Verifier)
- Manage Suras, Translations, Interpretations
- About Us, Author, Contact Us, Help pages
- Audit logging

## Setup
1. Copy .env.example to .env.local
2. Update MongoDB URI and login codes
3. Run npm install
4. Run npm run dev
